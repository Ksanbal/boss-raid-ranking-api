## 🚀 프로젝트 실행

### 1. .env 작성

```text
MODE="prod" # dev, prod
PORT=3000
```

### 2. 프로젝트 실행

```bash
# npm
$ npm install

$ npm build; npm run start:prod

# yarn
$ yarn

$ yarn build; yarn start:prod
```
